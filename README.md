# Go Business Referral Dashboard

A React + Vite application built for the Go Business Referral Dashboard assessment.

## Features

- Authentication
- Protected Routes
- Referral Dashboard
- Search & Sort
- Client-side Pagination
- Referral Details Page
- Responsive Design

## Tech Stack

- React
- Vite
- React Router
- Axios
- js-cookie
- Ant Design

## Run Locally

npm install
npm run dev

## Build

npm run build
